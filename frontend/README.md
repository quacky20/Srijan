# Srijan 2026 - Frontend

This project is built with React and Vite.

## Available Scripts

In the project directory, you can run:

### `npm install`

Installs the required dependencies for the project.

### `npm run dev`

Runs the app in the development mode.<br />
Open [http://localhost:5173](http://localhost:5173) (or another port if 5173 is in use) to view it in your browser.

The page will reload when you make changes.<br />
You may also see any lint errors in the console.

### `npm run build`

Builds the app for production to the `dist` folder.<br />
It correctly bundles React in production mode and optimizes the build for the best performance.

The build is minified and the filenames include the hashes.<br />
Your app is ready to be deployed!

### `npm run preview`

Serves the production build locally to preview it before deployment.

### `npm run lint`

Lints the project files for any code quality issues.
